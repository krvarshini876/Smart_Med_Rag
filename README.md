SmartMed-RAG is an Explainable Retrieval-Augmented Generation (RAG) based medical assistant that provides symptom analysis, healthcare recommendations, drug intelligence, and nearby hospital recommendations using NLP and semantic retrieval techniques.

 Features
Symptom-based health assessment
Retrieval-Augmented Generation (RAG)
Semantic medical knowledge retrieval
Drug information and recommendations
Nearby hospital recommendations
Explainable AI-powered responses
Modular and scalable architecture
